BiCute 스킨은 인기 스킨 Odyssey의 장점을 계승하면서, 다양한 문제점을 보완하고 사용자 맞춤 기능을 강화한 티스토리 블로그용 스킨입니다.

 

💡 주요 특징

✅ Odyssey의 장점 계승

 

  • PC 레이아웃 : 2단 우측 사이드바 / 1단 서랍형 사이드바
  • 목록 탐색 타입 : 페이지 나열 / 더보기 버튼
  • **티스토리 '메뉴'**를 활용한 네비게이션 바 적용 가능
  • 헤더 슬로건 / 이벤트 배너 섹션 지원
  • 사이드바 : 프로필 + SNS 링크(YouTube, Instagram, Twitter)
    ※ Facebook은 제거됨
  • 트위터 타임라인 적용 가능
  • 푸터 : 외부링크 최대 3개 / 푸터 문구 2줄 삽입 가능
  • 티스토리 링크 기반 관련 사이트 연결 기능

 

 

🛠 Odyssey의 문제점 보완

  • 반응형 사이즈 어긋남 해결
  • 헤더 범위 튀는 현상 제거
  • HTML 속성값 누락 문제 수정
  • 푸터에서 클릭만 되고 보이지 않던 링크 제거
  • 댓글 하단의 더보기 버튼 오류 수정

 

🎨 디자인 및 편의 기능 개선

  • 둥근 모서리 카드형 디자인 적용 (썸네일 등 요소)
  • 마우스 포인터 스타일 커스터마이징
  • 스크롤 진행도 표시 프로그레스 바 적용
  • 자동 목차 생성 (제목1 + 제목2 기반)
  • 라이트 / 다크 모드 지원
  • 텍스트 스타일 커스터마이징
    (제목1, 제목2, 제목3, 굵게, 기울임, 밑줄 등)
  • 코드블럭 및 클립보드 복사 기능 추가
  • 그 외 지속적인 기능 업데이트 예정

 

🧾 커버 스타일 (총 4가지)

1. 베이직 타입

  • 제목 + 본문 + 우측 대표 이미지
  • 게시물 수만큼 아래로 나열

2. 16:9 비율 1단 메인

  • 가로형 대표 이미지 + 제목 + 본문
  • 4개 이상일 경우 ‘더보기’ 버튼 제공

3. 1:1 비율 3단 포스터

  • 정사각형 썸네일 + 제목
  • 격자 형태로 나열

4. 단순 텍스트 타입

  • 이미지 및 본문 없이 제목만 표시
  • 공지사항 등 간단한 글에 적합
