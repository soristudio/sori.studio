version_1.4.0_2023.10.17

신규 기능 추가: My Cards Database

  • 덱 빌드 기능과는 별개로 독자적으로 돌아가는 유틸리티 페이지입니다.
    사용 용도는 자신이 가진 모든 카드의 개수를 기록하여 거래 및 자신의 카드 관리에 도움이 되도록 기록하여 사용하는 곳입니다.

    덱 빌더 페이지와는 다음과 같은 차이가 있습니다
    – 4장 장수 제한이 없습니다.
    – 페러랠 카드도 표시됩니다. (현재는 코드기어스와 샤이니 컬러즈만 적용)
    – 덱으로 완성하여 출력하는 기능은 없습니다.

    이곳에 기록한 모든 데이터는 자신의 브라우저 내부 캐쉬로 저장됩니다.
    즉, 캐쉬를 지우지 않는다면 기록이 남아있습니다만… 캐쉬를 지울 경우 기록 또한 사라집니다.
    그렇기에 데이터 백업 기능을 추가하였습니다.
    백업 버튼을 누를경우 작성한 카드 기록이 파일로 저장되며, 언제든지 데이터 복구를 눌러 해당 파일을 업로드 하면 기존의 기록이 복구되어 이어서 사용할 수 있습니다.

version_1.3.0_2023.10.12

신규 작품 추가

  • UA13ST, UA13BT - 블루 록(ブルーロック)

일반

  • 작품 선택 방식 변경
    작품을 선택하는 형태가 변경되었습니다.
  • 메뉴 추가
    타이틀 선택, 덱 기록 불러오기, 언어 변경등 앞으로 생길 기능들은 메뉴를 통해 설정 가능합니다.
  • 다국어 지원
    기본 언어는 일본어로 선택되어있지만, 언어 선택을 통하여 한국어, 영어, 일본어로 사용 가능합니다.
    선택한 언어는 브라우저에 저장되어 캐쉬를 삭제하기 전까지는 재접속 하더라도 선택한 언어로 자동 변환됩니다.
  • 히스토리 기능 / 핀(고정) 기능 추가
    자신이 작성하여 저장까지 마친 덱 리스트는 히스토리 내역에 자동으로 기록됩니다.
    최대 20개까지 기록되며, 제한 수량을 넘을경우 오래된 기록부터 삭제됩니다.
    핀 기능을 이용하여 기록을 선택하게되면 고정된 기록은 삭제되지 않습니다. 20개가 넘을 경우에도 핀이 찍혀있지 않는 기록부터 삭제됩니다.
    이 기능도 사용자의 브라우저에 기록되어 캐쉬를 삭제할때까지 유지됩니다.
